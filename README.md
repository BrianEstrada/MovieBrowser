# MovieBrowser
An Android Sample Application for browsing TheMovieDB

## About the App

The Application uses the following Libraries
- Retrofit
- Dagger 2
- RxJava 2
- Android Support Design
- Recycler View Library
- Card View Library
- Glide
- OkHttp
- Gson

The Library is written using the MVP Architecture.
